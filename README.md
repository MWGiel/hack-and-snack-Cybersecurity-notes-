## Lessons Learned
- Early-stage NFT projects often have well-intentioned setups but can miss minor security details.
- Responsible disclosure is essential: provide clear, actionable recommendations without creating risk.
- Monitoring and reviewing Discord server security is a valuable skill for Web3 community projects.
